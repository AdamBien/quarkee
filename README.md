# quarkee

opinionated quarkus.io with Java EE-stic out-of-the-box dependencies

# create a project

```
mvn archetype:generate -DarchetypeGroupId=com.airhacks -DarchetypeArtifactId=quarkee-archetype -DarchetypeVersion=0.0.1
```
